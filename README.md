## 🎯 About

Dyranis Consultancy is an **IRA-licensed independent insurance advisory firm** helping individuals, families, and businesses across Kenya navigate the complex world of insurance. ## 📦 Installation

### Prerequisites
- Node.js 18+ or Bun
- npm, yarn, or bun package manager

### Setup

```bash
# Clone the repository
git clone https://github.com/yourusername/dyranis-consultancy.git
cd dyranis-consultancy

# Install dependencies
npm install
# or
bun install

# Start development server
npm run dev
# or
bun run dev

# Build for production
npm run build
# or
bun run build

# Preview production build
npm run preview
# or
bun run preview
```

The site will be available at `http://localhost:5173`

## 🧮 Quote Calculator

The quote calculator allows users to:
1. Select an insurance service
2. Choose coverage amount (KES 100,000 - 5,000,000)
3. Select coverage tier (Basic, Standard, Premium, Elite)
4. Get instant premium estimate
5. Share quote via WhatsApp

**Base Rate:** KES 5,000  
**VAT:** 16%  
**Multipliers:** 1.0x - 2.5x based on tier
